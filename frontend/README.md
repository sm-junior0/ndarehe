# NDAREHE Explorer Hub

A comprehensive travel and accommodation platform for exploring Rwanda's hidden gems and authentic local experiences.

## About the Project

NDAREHE Explorer Hub is a modern web application designed to help travelers discover and book authentic accommodations, plan airport pickups, and explore local experiences in Rwanda. The platform provides a seamless interface for connecting travelers with local hosts and experiences, enhancing Rwanda's tourism sector.

## Features

- **Accommodation Booking**: Find and book hotels, homestays, and unique accommodations across Rwanda
- **Airport Pickup Service**: Reliable transportation from Kigali International Airport
- **Local Experiences**: Discover city tours, cultural experiences, and vibrant nightlife
- **Trip Planning**: Get personalized recommendations from local experts
- **Responsive Design**: Modern, mobile-friendly interface built with React and Tailwind CSS
- **Multi-language Support**: Ready for international travelers
- **Secure Booking System**: Safe and reliable payment processing

## Technologies Used

This project is built with modern web technologies:

- **Frontend Framework**: React 18 with TypeScript
- **Build Tool**: Vite for fast development and optimized builds
- **Styling**: Tailwind CSS for utility-first styling
- **UI Components**: shadcn/ui for beautiful, accessible components
- **Routing**: React Router for seamless navigation
- **Icons**: Lucide React for consistent iconography

## Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd ndarehe-explorer-hub
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`
